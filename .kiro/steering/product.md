# Product Overview

Scrapiz Admin is a comprehensive admin dashboard for managing a scrap pickup and recycling marketplace. The platform connects sellers, buyers, and pickup agents to facilitate scrap collection and recycling operations.

## Core Functionality

- **Order Management**: Create, track, and manage scrap pickup orders with details like category, weight, pricing, and status
- **User Management**: Handle multiple user roles (sellers, buyers, agents, admins, superadmins) with KYC verification and document management
- **Agent Operations**: Automatic agent assignment to orders based on location using AI-powered Cloud Functions
- **Pricing System**: Dynamic scrap pricing per category with historical tracking
- **Payment Processing**: Process payouts to sellers and agents, track transactions, generate invoices
- **Analytics & Reporting**: Generate reports on orders, revenue, scrap volume, and agent performance
- **Role-Based Access Control**: Restrict dashboard features based on user roles

## Design Philosophy

- Clean, trust-inspiring interface with environmental awareness theme
- Primary color: Soft Blue (#A0D2EB) for trust and cleanliness
- Background: Light Gray (#F0F4F8) for readability
- Accent: Warm Orange (#F2BE22) for CTAs
- Typography: Inter font family for modern, neutral aesthetic
- Card-based layouts for summaries, tables for detailed data
- Subtle transitions and animations for smooth UX
