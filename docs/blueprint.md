# **App Name**: Scrapiz Admin

## Core Features:

- Pickup Order Management: Create, view, update, and delete scrap pickup orders with details like seller, agent, category, weight, and address.
- User Management: Manage user accounts (sellers, buyers, agents, admins) with roles, KYC status, documents, and contact information.
- Agent Assignment: Automatically assign pickup agents to orders based on location using a Cloud Function tool.
- Pricing Management: Set and update scrap pricing per category to reflect market rates.
- Payment Processing: Process payments to sellers and agents, track transaction history, and generate invoices.
- Reporting and Analytics: Generate reports on orders, revenue, scrap volume, and agent performance.
- Role-Based Access Control: Implement role-based access to restrict dashboard functionalities based on user roles (Super Admin, Admin, Support Staff, Agent).

## Style Guidelines:

- Primary color: Soft Blue (#A0D2EB) to create a sense of trust and cleanliness, aligning with recycling and environmental awareness.
- Background color: Light Gray (#F0F4F8) to provide a clean and neutral backdrop, ensuring that data is easily readable.
- Accent color: Warm Orange (#F2BE22) to highlight key actions and calls to action.
- Body and headline font: 'Inter', a sans-serif font, provides a modern, machined, objective, neutral look.
- Use simple, clear icons from Material UI to represent different categories and actions.
- A clean and organized layout with cards for summaries and tables for detailed data views, and a sidebar navigation for different sections.
- Subtle transitions and animations on data updates and page transitions to provide a smooth user experience.
